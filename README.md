<h2><a id="user-content-features" class="anchor" aria-hidden="true" href="#features"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M7.775 3.275a.75.75 0 001.06 1.06l1.25-1.25a2 2 0 112.83 2.83l-2.5 2.5a2 2 0 01-2.83 0 .75.75 0 00-1.06 1.06 3.5 3.5 0 004.95 0l2.5-2.5a3.5 3.5 0 00-4.95-4.95l-1.25 1.25zm-4.69 9.64a2 2 0 010-2.83l2.5-2.5a2 2 0 012.83 0 .75.75 0 001.06-1.06 3.5 3.5 0 00-4.95 0l-2.5 2.5a3.5 3.5 0 004.95 4.95l1.25-1.25a.75.75 0 00-1.06-1.06l-1.25 1.25a2 2 0 01-2.83 0z"></path></svg></a>Features</h2>
<p>Job Search App project aims create an app using API with React-Native for job search. This app comprises use of React-Navigation/Stack, Axios, Async-Storage, Webview, Modal and Vector-Icons.</p>
<ul>
<li>When the user lands the Home page he/she can see list of programming language cards which consists job postings each of them in advance.</li>
<li>User can scroll tabs horizontal and check list of programming languages, when tap on a language button, job postings are displayed.</li>
<li>User can search any job with using SearchBar. Job posting blocks are clickable and when user clicks on a job posting, modal is appeared. User can display details and it can be saved in memory if desired.</li>
<li>User can check saved jobs with using button where placed bottom-right on page. On this page, it is allowed delete and check details of saved jobs.</li>
</ul>

<p>To use this application, the packages for Navigation/Stack, Axios, Async-Storage, Webview, Modal and Vector-Icons should be installed/identified with prior to running the app. From your command line:</p>
<pre><code>npm install @react-navigation/native
For expo :
expo install react-native-gesture-handler react-native-reanimated react-native-screens react-native-safe-area-context @react-native-community/masked-view
For bare React-native:
npm install react-native-reanimated react-native-gesture-handler react-native-screens react-native-safe-area-context @react-native-community/masked-view

npx pod-install ios(for IOS)

npm install @react-navigation/stack

npm install react-native-modal

npm install react-native-vector-icons

npm install @react-native-async-storage/async-storage

npm install react-native-webview

</code></pre>
